ARCHIBUS/FM View Tests


To run the view tests:
(1) Edit the strUrlRoot variable in load-test-preferences.js file to reflect the location of your server.
(2) Log in to an ARCHIBUS/FM v14i user session.
(3) Load this load-test.htm file in the Address bar.
(4) Press the button below.
To re-run the test, press F5 to reload the page, then press start again.
