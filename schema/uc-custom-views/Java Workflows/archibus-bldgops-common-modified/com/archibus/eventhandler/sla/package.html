The SLA package contains all classes for handling Service Level Agreements.

The ServiceLevelAgreementHandler contains all Workflow Rules (Event Handlers) called from view files, for retrieving and saving SLA's. Other java (eventhandlers and manager) classes can use the ServiceLevelAgreement class directly for retrieving the SLA to be applied.
The Service Window is used to calculate escalations.

The CalendarManager contains helper functions for holidays.
